# tablemultii
# Table de multiplication en Python

## Description

Ce programme permet d'afficher la table de multiplication d’un nombre choisi par l'utilisateur.

C’est mon premier projet Python réalisé après environ 3 semaines d’apprentissage.

## Fonctionnalités

- Demande un nombre à l'utilisateur
- Affiche la table de multiplication
- Fonctionne avec des nombres entiers

## Comment utiliser

1. Télécharger le fichier Python
2. Ouvrir un terminal
3. Lancer la commande :

python table.py

4. Entrer un nombre
5. La table de multiplication s'affiche

## Exemple d'utilisation

Entrez un nombre : 5

5 x 1 = 5  
5 x 2 = 10  
5 x 3 = 15  
...
